# E-commerce-Sales-Dashboard-Excel
Interactive Excel dashboard analyzing e-commerce sales, customer behavior, and channel performance.

## 🔍 Project Overview
This project is an interactive Excel dashboard analyzing e-commerce sales data. It provides insights into sales trends, customer behavior, and channel performance.

## 📌 Key Features
- KPI Cards (Sales, Orders, AOV, Customers)
- Monthly Sales Trend Analysis
- Sales by Channel
- Top 5 States by Sales
- Customer Demographics (Age & Gender)
- Interactive Slicers

## 💡 Key Insights
- Peak sales observed in March
- Amazon is the top-performing channel
- Women contribute more to overall sales
- Maharashtra generates the highest revenue

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
